## 1. Client ask (verbatim)
> The portal feels flaky and I'm always the last to hear about it — usually a patient complains
> before I even know there's a problem. I want real visibility so I can get ahead of issues. And
> that patient AI summary you stood up makes me nervous now that it's live; I need to know it can't
> say something off about a patient. Here's a week of logs and a transcript from it that honestly
> concerned me.

## 2. What it really is
- Two halves: (a) **observability** — structured tracing + golden-signals dashboard + one working
  SLO alert; (b) **LLM output guardrail** — the "transcript that concerned me" is a **hallucinated
  clinical claim** (e.g. "continue metformin" for a no-meds patient) reaching a clinician unchecked.
  Guardrail = grounding check + validation + refuse-unknown.

## 3. Scope
### In scope
- Structured tracing across the request path.
- Golden-signals dashboard (latency / traffic / errors / saturation).
- One **working** SLO alert (fires on a real breach condition).
- LLM guardrail: output grounded in retrieved facts; validate; refuse when unknown.
### Explicitly OUT (owning week)
- Safe-Harbor de-id + BAA governance → W8 (this week is grounding/safety, not vendor-disclosure).
- Append-only audit trail / breach-detection report → W10.

## 4. Deliverables
1. Structured tracing instrumentation.
2. Golden-signals dashboard.
3. One SLO alert (proven to fire).
4. LLM output guardrail (grounding + validation + refuse-unknown) with tests.

## 5. Acceptance criteria (done-definition)
- [ ] Trace spans cover the LLM call path and the records path; a trace is viewable end-to-end.
- [ ] Dashboard shows the four golden signals from real data.
- [ ] One SLO alert is defined AND demonstrated firing on a breach (not just configured).
- [ ] Guardrail **refuses / flags** an ungrounded claim — reproduce the concerning transcript
      (no-meds patient) and show the guardrail blocks the hallucinated med recommendation.
- [ ] Guardrail has an **adversarial** test (a plausible-but-ungrounded claim it must reject).

## 6. Landmines (CLAUDE.md §9)
- ⚠️ AI output guardrail — LLM summary is ungrounded; hallucinated clinical content reaches clinicians
  unchecked. Safety + liability.
- ⚠️ Observability must stay **metadata-only / no PHI** to the tracing vendor (see ADR 0006 direction).
